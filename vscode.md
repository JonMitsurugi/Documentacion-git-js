# Atajos

## VS Code

* Comentar una linea : <kbd>Ctrl</kbd> + <kbd>}</kbd>.

* Comentar varias lineas : <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>A</kbd>.

* Ocultar / mostrar Panel de Terminal : <kbd>Ctrl</kbd> + <kbd>J</kbd>.

## Windows 10

* Captura de pantalla : <kbd>Shift</kbd> + <kbd>&#8862; Win</kbd> + <kbd>S</kbd>.

* Abrir Windows Game Bar : <kbd>&#8862; Win</kbd> + <kbd>G</kbd>.

* Ocultar / mostrar Panel de Terminal : <kbd>&#8862; Win</kbd> + <kbd>Alt</kbd> + <kbd>R</kbd>.
